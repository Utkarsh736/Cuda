## 🔧 Setup & Compilation
### Requirements
- NVIDIA GPU with CUDA support
- CUDA Toolkit installed
- GCC or Clang compiler

### Compilation & Execution
To compile a CUDA program:
```sh
nvcc -o my_program my_program.cu
./my_program
```
To profile execution:
```sh
nvprof ./my_program
```
